忘记了 root 密码怎么办？

  1. 修改配置文件，取消密码验证
  2. 重启mysql
    sudo service mysql restart
  3. 登录mysql, 修改密码
  4. 重新编辑配置文件， 启用密码验证。
  5. 重启 mysql 服务。
    sudo service mysql restart
  6. 重新用 root 用户登录 mysql
    sudo mysql -u root -p
